아래 순서대로 따라해주세요.

1단계 — console.firebase.google.com 접속 후 구글 로그인
2단계 — "프로젝트 만들기" → 이름 입력 → Analytics 끄고 완료
3단계 — 왼쪽 메뉴 Firestore Database → "데이터베이스 만들기" → 테스트 모드
4단계 — ⚙️ 프로젝트 설정 → "앱 추가" → 웹(</>) → firebaseConfig 값 아래에 입력
apiKey
authDomain
projectId
값을 모두 입력해주세요.
